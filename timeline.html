Exploring the Building Blocks of the Web

01

What is HTML?

HTML (HyperText Markup Language) is the skeleton of every webpage. It provides the basic structure and content using a system of tags and elements.

  • Uses tags like <h1>, <p>, <div> to structure content
  • The latest version is HTML5 with new semantic elements
  • Works together with CSS and JavaScript
02

What is CSS?

CSS (Cascading Style Sheets) makes HTML look beautiful and stylish with colors, layouts, and animations.

.fancy-button {
        background: linear-gradient(45deg, #ffb4d9, #ffd6e7);
        border-radius: 25px;
        box-shadow: 0 4px 15px rgba(0,0,0,0.1);
        transition: all 0.3s ease;
    }
    
    .fancy-button:hover {
        transform: translateY(-3px);
        box-shadow: 0 6px 20px rgba(0,0,0,0.15);
    }

Building a Room

I imagined my website as a cute little room! Each section represents a different area:

  • The entrance is the welcome page (with window and cat)
  • The timeline is a calendar on the wall
  • The CSS study area is a dressing table (for "beautifying")

This design makes the website feel cozier, like entering a little world.
